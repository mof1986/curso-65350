Descripción del simulador: Simulador de compras

Este simulador permite a los usuarios seleccionar productos (en este caso, de construcción), agregarlos a un carrito de compras y visualizar el resumen con precios calculados según las cantidades seleccionadas. El menú principal ofrece tres opciones: elegir/modificar productos, ver el carrito y salir. El simulador guía al usuario a través de estas opciones, interactuando mediante cuadros de diálogo.
